Edit Trip Details:

1. From the Trip Details page, the user changes the name and times of the selected Trip.
2. The user selects the Edit button to confirm the change.
3. The Trip is edited and the user is returned to the Home page where the trip's new information is now displayed.

3a. Invalid Info (Duplicate name, overlapping times, end date before start date)
	- The trip is not changed and an error message appears.
