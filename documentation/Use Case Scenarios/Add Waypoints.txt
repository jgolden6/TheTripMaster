Add Waypoints:

1. From the Trip Details page, the user selects the Add Waypoint button.
2. The Add Waypoint page is opened, displaying fields for name, start date and time, and an optional end date and time.
3. The user selects confirm to add the waypoint.
4. The waypoint is created. The user is returned to the Trip Details page, where the waypoint is now displayed in the  waypoint list.


3a. The user selects cancel to abort the Add.
	- The user is returned to the Trip Details page. No waypoint is added.


4b. Invalid Info (Duplicate name, overlapping times, end date before start date)
	- The waypoint is not created and an error message appears.
